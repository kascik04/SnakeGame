SNAKE GAME PROJECT - DỰ ÁN GAME RẮN SĂN MỒI
Dự án cá nhân | Java | [Năm thực hiện]

Phát triển game Snake toàn diện với giao diện chuyên nghiệp sử dụng Java Swing. Tích hợp hệ thống độ khó đa cấp (Dễ/Trung bình/Khó), nhiều kiểu dáng rắn tùy chỉnh, hệ thống âm thanh đa phương tiện, power-up làm chậm game, và chướng ngại vật tường động. Ứng dụng kỹ năng lập trình hướng đối tượng, quản lý luồng cho game loop, xử lý sự kiện và thiết kế giao diện người dùng chuyên nghiệp.

Công nghệ sử dụng: Java Swing, Audio Programming, Thread Management, OOP Design Patterns, Graphics Programming
